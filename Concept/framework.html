Overview

The Global Framework includes essential functions for managing hosts, functions, and timers. This documentation provides details for the following core functionalities:

  • Host Management: register_host, gHost, getDefaultHost, registerAllHosts
  • Function Management: register_gFunc, gFunc

Host Management

register_host

Creates a global tie-in for processors and Secure Platform devices.

def register_host(host_name, device_class='Processor', description='', header='', text_return=None):
    """Register a host for global access."""

gHost

Retrieve a registered host from the global database.

def gHost(host_to_retrieve):
    """Retrieve a host by its name."""

getDefaultHost

Retrieve the default UIHost from the configuration.

def getDefaultHost(return_as_alias=False):
    """Get the default host or its alias."""

registerAllHosts

Register multiple hosts using a list of host tuples.

def registerAllHosts(list_of_host_tuples, text_return=None):
    """Register all specified hosts."""

Function Management

register_gFunc

Registers a function globally for recall from any function file.

def register_gFunc(function, description=''):
    """Register a global function."""

gFunc

Call a globally registered function by its ID.

def gFunc(function_id, *args, **kwargs):
    """Invoke a global function."""
