Framework Essentials

Framework Essentials

Overview

The SP Framework provides a structured approach to managing AV control systems, ensuring consistency and scalability. This section covers essential functions and their descriptions, categorized for ease of reference.

Host Management

FunctionDescription
register_hostCreates a global tie-in for processors and SPDs.
gHostRetrieves a host from the host database.
getDefaultHostRetrieves the default UIHost from the configuration.
registerAllHostsRegisters all hosts defined in the sandbox configuration.
anyUI_HostRetrieves the UI host from the tlpDB based on the provided ui_name.

Global Functions

FunctionDescription
register_gFuncCreates a global tie-in for functions, allowing them to be recalled from any function file.
gFuncCalls a global function with the given ID and passes in arguments and keyword arguments.
gFuncLoaderLoads a global function from one file (including arguments & kwargs) and executes it in another.

Global Timers

FunctionDescription
register_gTimerCreates a global tie-in for Extron Timer Class.
gTimerCalls a global timer.

UI Hosts / TLPs

FunctionDescription
TLP_LoaderLoads a series of TLP Panels by alias.
getDefaultUIHostRetrieves the default UIHost (TLP) from the framework configuration.
getUIHostAliasReturns the device alias of a button's TLP (UIHost).

Debugging & Console Tools

FunctionDescription
dvTracePrints to the standard print window and to a TCP port for debugging, formatting, and filtering.
consoleWindowProvides a scrollable text window for reviewing logs and debugging messages.
rolling_text_scrollScrolls text for boot-up, auto-dumping old lines when a threshold is met.
BootTimerPunchInRecords the start time of a boot timer for a given page.
BootTimerPunchOutRecords the completion of the boot tracking process.

UI Element Management

FunctionDescription
MultiButtonMakeCreates a button with the specified name on each desired UIHost panel in the tlpDB.
MultiLevelMakeCreates a level control on all UIHost panels where applicable.
MultiSliderMakeCreates a slider control on all UIHost panels where applicable.
MultiLabelMakeCreates a label on each UIHost panel.

ME Set Tools

FunctionDescription
LocalMESetCurrentSets the currently pressed MESet to "Set Current" based on the provided button object.
MultiMESetCurrentIterates through the meDB and sets all MESets with the specified name to the current button.
MultiMESetStatesSets the state for buttons within specified MESets across various panels.
meDB_GetCurrentRetrieves the current button object from the specified MESet in the meDB.

Page & Popup Navigation

FunctionDescription
MultiShowPageShows a specified page on all panels within the given UI hosts.
MultiShowPopupDisplays a popup on the specified UI hosts.
MultiHidePopupHides a specified popup on selected UI hosts.
MultiHideAllPopupsHides all popups across defined UI hosts.
MultiHidePopupGroupHides a specific group of popups across UI hosts.

Sound Management

FunctionDescription
MultiPlaySoundPlays a specified sound on all panels within the given UI hosts.
